Instructions to Replace Logo

  1. Take a screenshot of the preview below at exactly 1200x630 pixels
  2. Open in image editor (Photoshop, GIMP, Canva, etc.)
  3. Replace the placeholder logos (dashed border boxes) with your actual Gialoma logo
  4. Maintain the size and position - main logo should be ~70x70px, URL logo ~20x20px
  5. Save as PNG and name it gialoma-social-preview.png
  6. Upload to your public folder and update your meta tags to use the PNG version

Tip: Use browser dev tools to capture the exact dimensions, or use this HTML file in a screenshot tool.
